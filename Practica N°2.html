Práctica_N°2 Etiquetas HTML que no estaban en clases

1. <article>

Noticia independiente

Este artículo contiene un contenido propio y autónomo.

2. <section>

Sección temática

Sirve para dividir la página en bloques organizados por tema.

3. <aside>

Contenido lateral, como publicidad o enlaces relacionados.

4. <footer>

© 2025 - Todos los derechos reservados

5. <mark>

Este texto tiene una palabra resaltada.

6. <abbr>

CSS se usa para dar estilo a las páginas web.

7. <address>

Escrito por Juan Pérez.
Correo: juan@example.com

esto es información de contacto
nunca se debe usar "address" para otros textos que no sean información de contacto

8. <audio>

Sirve para reproducir archivos de audio en la página web,

9. <video>

Sirve para reproducir archivos de video en la página web,

10. <canvas>

Tu navegador no soporta canvas.

11. <details>

Haz clic para expandir

Este es el contenido oculto dentro de la etiqueta <details>.

12. <summary>

Resumen visible

El texto dentro del summary es lo que se ve como título.

13. <meter>

Nivel de batería: 70%

14. <output>

+ = 8

15. <picture>

La etiqueta picture permite mostrar imágenes diferentes según el tamaño de la pantalla.
